Homework_17.ipynb - решение задачи регрессии на датасете Wallmart.csv методом случайного леса. Исправлен код по сравнению с исходным от 30.01.2025, точность предсказания около 6.7%

Homework_22_CNN.ipynb -  попытка решить задачу регрессии с помощью сверточной нейронной сети. Точность плохая (отрицательный R^2).

Homework_22_DenseNN_v1.ipynb - решение задачи регрессии полносвязной нейронной сетью. Нет предобработки данных. Точность плохая (отрицательный R^2).

Homework_22_DenseNN_v2.ipynb - данные нормированы стандартным скейлером, в том числе номер магазина и бинарная колонка Holiday_Flag. Средняя точность неплохая 7.6%.

Homework_22_DenseNN_v3.ipynb - улучшенная обработка параметров. Бинарные данные (колонка Holiday_Flag) никак не обрабатываются. Номер магазина обработан с помощью OneHotEncoder. Остальные колонки нормированы стандартным скейлером. Средняя точность предсказания около 6.7%, как у случайного леса.
